Politica de Privacidade - DriverStatus

Ultima atualizacao: 27 de abril de 2026

Resumo

O DriverStatus respeita sua privacidade. O aplicativo analisa os drivers do seu computador localmente, sem enviar dados para servidores externos.

Dados coletados

Nenhum dado pessoal e coletado, armazenado ou transmitido.

  • Drivers: A lista de drivers e obtida localmente via Windows SetupAPI. Nenhuma informacao e enviada para servidores.
  • Backup: Os backups de drivers sao salvos localmente no seu computador, na pasta que voce escolher.
  • Configuracoes: Salvas localmente no seu computador (pasta AppData).

Conexao com a internet

O aplicativo acessa a internet apenas para:

  • Verificar se ha atualizacoes disponiveis (consulta um arquivo JSON no GitHub).
  • Baixar atualizacoes quando disponiveis.

Nenhum dado pessoal e transmitido nessas conexoes.

Funcionalidade de IA

O recurso de diagnostico assistido por IA copia um resumo tecnico dos drivers para a area de transferencia. Voce decide se e quando colar esse texto em um servico de IA externo. O DriverStatus nao envia dados automaticamente para nenhum servico de inteligencia artificial.

Permissoes

  • Acesso ao sistema: Usado para consultar informacoes de drivers via Windows SetupAPI.
  • Windows Update: Usado para verificar atualizacoes de drivers disponiveis pelo Windows Update.
  • Administrador (opcional): Solicitado sob demanda apenas para operacoes que exigem privilegios elevados (backup/restauracao de drivers).

Terceiros

O DriverStatus nao compartilha dados com terceiros. Nao utiliza analytics, rastreamento ou publicidade.
